Тестове завдання:
Написати програму для валідації даних.

Приклади даних, які будемо валідувати:
- Серія та номер паспорта: СЮ112233
- РНОКПП: 1122334455 (10 цифр) 
- Дата народження: 01.01.2001
- Номер пристрою: B5A648BA-262F-47BF-A0F5-4FDF68AC0C12 (iOS) та 3614435a-8b46-42de-a7b5-bf1d72c4b913 (Android)

Валідація має відбуватись за допомогою RegEx

Програма має повертати True або False, в залежності від валідності даних.

Код повинен мати можливості для розширення, в майбутньому можливе додавання нових даних для валідації.

Приклад виклику програми:
> validation.exe rnokpp 1234567890
> True



--------------------------------------------------------------------------------------------------------------------------

1) Серія та номер паспорта - у заданому форматі і без використання "ь"
2) дата народження - рік від 1800 до 2099
3) команди для різних типів даних: "rnokpp", "passport", "birth_date", "device_number"
	Наприклад:
	> validator.exe rnokpp 1234567890
	> validator.exe birth_date 30.12.2023

4) Якщо параметри задані не коректно або відсутні, результат програми буде False.
5) РНОКПП - будь-які 10 цифр без обмежень.
